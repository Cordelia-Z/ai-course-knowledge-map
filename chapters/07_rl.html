二、层级知识结构

按“一级知识点 → 概念/公式/例子/重点/易错/考点”整理。

1. 强化学习问题

智能体不知道完整环境模型，只能边行动边学习。

关键概念

  • 未知 T/R：转移概率和奖励函数无法预先给出。
  • 交互经验：每步得到 s,a,r,s'。
  • episode：从起点到终止的一条轨迹。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 机器人在未知地图中试探并学习避障。

老师强调点

  • RL 是为了更好地行动，不只是预测。

易错点

  • 易错：把 RL 当成普通监督学习。

考试可能考法

  • 解释 RL 与 MDP 关系。

2. Model-based Learning

先估计环境模型，再用规划算法求策略。

关键概念

  • Monte Carlo：用完整采样轨迹回报估计。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 走多次某路线后统计成功率和平均收益。

老师强调点

  • model-based 先学模型再规划。

易错点

  • 易错：model-based 不等于直接更新 Q。

考试可能考法

  • 区分 model-based 与 model-free。

3. Model-free Learning

不显式估计 T/R，直接学习价值。

关键概念

  • TD learning：用当前奖励和下一状态估计修正当前价值。

公式/规则

  • Q(s,a)←(1−η)Q(s,a)+η[r+γ max_a'Q(s',a')]

典型例子

  • 发现某动作奖励差，就降低该状态-行动 Q。

老师强调点

  • Q-learning 是高频计算题。

易错点

  • 易错：更新的是状态-行动对。

考试可能考法

  • 手算 Q-learning 更新。

4. 探索与利用

在尝试新行动和选择当前最优之间平衡。

关键概念

  • exploration：尝试不确定行动，获得新信息。
  • exploitation：选择当前估计最好的行动。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 常去熟悉餐厅是利用，尝试新店是探索。

老师强调点

  • 探索不是犯错，是学习未知环境的必要成本。

易错点

  • 易错：只利用会导致认知不足。

考试可能考法

  • 解释探索-利用权衡。

# Lecture 8：MDPs II / Reinforcement Learning

本章核心主题：本章在 MDP 基础上进入强化学习：当 T/R 未知时，智能体通过交互经验学习模型或价值。

老师重点：强化学习不是没有 MDP，而是 MDP 的转移和奖励未知。

## 1. 强化学习问题

### 1.1 二级知识点

- 核心概念：智能体不知道完整环境模型，只能边行动边学习。
- 公式/规则：本节以概念/流程为主
- 典型例子：机器人在未知地图中试探并学习避障。
- 老师重点：RL 是为了更好地行动，不只是预测。
- 易错点：易错：把 RL 当成普通监督学习。
- 考点：解释 RL 与 MDP 关系。

### 1.2 未知 T/R

- 类型：核心概念
- 说明：转移概率和奖励函数无法预先给出。

### 1.3 交互经验

- 类型：核心概念
- 说明：每步得到 s,a,r,s'。

### 1.4 episode

- 类型：核心概念
- 说明：从起点到终止的一条轨迹。

### 1.5 在线学习

- 类型：图示/流程
- 说明：行动产生数据，数据更新行为。

## 2. Model-based Learning

### 2.1 二级知识点

- 核心概念：先估计环境模型，再用规划算法求策略。
- 公式/规则：本节以概念/流程为主
- 典型例子：走多次某路线后统计成功率和平均收益。
- 老师重点：model-based 先学模型再规划。
- 易错点：易错：model-based 不等于直接更新 Q。
- 考点：区分 model-based 与 model-free。

### 2.2 估计转移

- 类型：图示/流程
- 说明：统计 s,a 到 s' 的频率。

### 2.3 估计奖励

- 类型：图示/流程
- 说明：用观察到的平均奖励估计 R。

### 2.4 Monte Carlo

- 类型：核心概念
- 说明：用完整采样轨迹回报估计。

### 2.5 再规划

- 类型：图示/流程
- 说明：估计模型后可运行价值迭代。

## 3. Model-free Learning

### 3.1 二级知识点

- 核心概念：不显式估计 T/R，直接学习价值。
- 公式/规则：Q(s,a)←(1−η)Q(s,a)+η[r+γ max_a'Q(s',a')]
- 典型例子：发现某动作奖励差，就降低该状态-行动 Q。
- 老师重点：Q-learning 是高频计算题。
- 易错点：易错：更新的是状态-行动对。
- 考点：手算 Q-learning 更新。

### 3.2 TD learning

- 类型：核心概念
- 说明：用当前奖励和下一状态估计修正当前价值。

### 3.3 Q-learning

- 类型：公式/推导
- 说明：直接更新 Q(s,a)，不需要 T。

### 3.4 TD target

- 类型：公式/推导
- 说明：r + γ max_a' Q(s',a')。

### 3.5 learning rate

- 类型：公式/推导
- 说明：控制新经验覆盖旧估计程度。

## 4. 探索与利用

### 4.1 二级知识点

- 核心概念：在尝试新行动和选择当前最优之间平衡。
- 公式/规则：本节以概念/流程为主
- 典型例子：常去熟悉餐厅是利用，尝试新店是探索。
- 老师重点：探索不是犯错，是学习未知环境的必要成本。
- 易错点：易错：只利用会导致认知不足。
- 考点：解释探索-利用权衡。

### 4.2 exploration

- 类型：核心概念
- 说明：尝试不确定行动，获得新信息。

### 4.3 exploitation

- 类型：核心概念
- 说明：选择当前估计最好的行动。

### 4.4 ε-greedy

- 类型：图示/流程
- 说明：多数时候利用，少数时候随机探索。

### 4.5 局部最优风险

- 类型：易错点
- 说明：只利用可能错过更好策略。

二、层级知识结构

按“一级知识点 → 概念/公式/例子/重点/易错/考点”整理。

1. 因子图与链式结构

图结构决定搜索和推理复杂度。

关键概念

  • 因子图复习：变量节点和因子节点交替连接。
  • 链式结构：变量按序连接，局部依赖明显。
  • 局部依赖：每个变量只与少量邻居直接相关。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 位置 Ct 只强依赖 Ct-1 和观测 Et。

老师强调点

  • 结构简单时可以动态规划式计算。

易错点

  • 易错：图结构不同，算法效率可能完全不同。

考试可能考法

  • 识别链式结构。

2. 贪婪、集束与局部搜索

在大空间中用启发式寻找较好解。

关键概念

  • 本项不作为单独节点展开。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • N 皇后可从随机完整赋值开始修补冲突。

老师强调点

  • 启发式快但不保证最优。

易错点

  • 易错：beam search 也可能丢掉潜在好路径。

考试可能考法

  • 比较三类搜索。

3. 条件化

固定部分变量，使剩余问题结构更简单。

关键概念

  • 复杂度权衡：枚举开销换取结构简化。
  • 割点直觉：固定关键变量可能切断图。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 固定中心变量后左右两部分独立。

老师强调点

  • 条件化是为了改变图结构。

易错点

  • 易错：不是随便固定变量。

考试可能考法

  • 解释 conditioning 的作用。

4. 变量消除

合并相关因子并消去变量。

关键概念

  • 中间因子：消除变量后产生的新因子。

公式/规则

  • newFactor(Y)=Σ_X ∏ factors(X,Y)

典型例子

  • 链式结构按顺序消除可避免巨大中间表。

老师强调点

  • 变量消除是结构化精确计算方法。

易错点

  • 易错：难点在消除顺序。

考试可能考法

  • 手推变量消除一步。

# Lecture 12：CSPs II

本章核心主题：本章扩展 CSP 的求解方法：链式结构、对象跟踪、贪婪搜索、集束搜索、局部搜索、条件化和变量消除。

老师重点：利用结构比盲目搜索更重要，链式结构和变量消除体现按结构计算。

## 1. 因子图与链式结构

### 1.1 二级知识点

- 核心概念：图结构决定搜索和推理复杂度。
- 公式/规则：本节以概念/流程为主
- 典型例子：位置 Ct 只强依赖 Ct-1 和观测 Et。
- 老师重点：结构简单时可以动态规划式计算。
- 易错点：易错：图结构不同，算法效率可能完全不同。
- 考点：识别链式结构。

### 1.2 因子图复习

- 类型：核心概念
- 说明：变量节点和因子节点交替连接。

### 1.3 链式结构

- 类型：核心概念
- 说明：变量按序连接，局部依赖明显。

### 1.4 对象跟踪建模

- 类型：图示/流程
- 说明：用时间序列变量表示目标变化。

### 1.5 局部依赖

- 类型：核心概念
- 说明：每个变量只与少量邻居直接相关。

## 2. 贪婪、集束与局部搜索

### 2.1 二级知识点

- 核心概念：在大空间中用启发式寻找较好解。
- 公式/规则：本节以概念/流程为主
- 典型例子：N 皇后可从随机完整赋值开始修补冲突。
- 老师重点：启发式快但不保证最优。
- 易错点：易错：beam search 也可能丢掉潜在好路径。
- 考点：比较三类搜索。

### 2.2 贪婪搜索

- 类型：图示/流程
- 说明：每步选当前最优局部选择。

### 2.3 集束搜索

- 类型：图示/流程
- 说明：每层保留若干较好候选。

### 2.4 局部搜索

- 类型：图示/流程
- 说明：从完整赋值出发局部修改。

### 2.5 局部最优

- 类型：易错点
- 说明：可能停在非全局最优解。

## 3. 条件化

### 3.1 二级知识点

- 核心概念：固定部分变量，使剩余问题结构更简单。
- 公式/规则：本节以概念/流程为主
- 典型例子：固定中心变量后左右两部分独立。
- 老师重点：条件化是为了改变图结构。
- 易错点：易错：不是随便固定变量。
- 考点：解释 conditioning 的作用。

### 3.2 选定变量

- 类型：图示/流程
- 说明：挑选变量枚举取值。

### 3.3 问题分解

- 类型：图示/流程
- 说明：固定后原图可能拆成子问题。

### 3.4 复杂度权衡

- 类型：核心概念
- 说明：枚举开销换取结构简化。

### 3.5 割点直觉

- 类型：核心概念
- 说明：固定关键变量可能切断图。

## 4. 变量消除

### 4.1 二级知识点

- 核心概念：合并相关因子并消去变量。
- 公式/规则：newFactor(Y)=Σ_X ∏ factors(X,Y)
- 典型例子：链式结构按顺序消除可避免巨大中间表。
- 老师重点：变量消除是结构化精确计算方法。
- 易错点：易错：难点在消除顺序。
- 考点：手推变量消除一步。

### 4.2 消除顺序

- 类型：图示/流程
- 说明：不同顺序导致中间因子大小不同。

### 4.3 合并因子

- 类型：公式/推导
- 说明：把含待消除变量的因子乘起来。

### 4.4 求和/最大化消去

- 类型：公式/推导
- 说明：根据任务对变量求和或取 max。

### 4.5 中间因子

- 类型：核心概念
- 说明：消除变量后产生的新因子。

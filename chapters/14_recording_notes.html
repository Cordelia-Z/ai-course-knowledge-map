二、层级知识结构

按“一级知识点 → 概念/公式/例子/重点/易错/考点”整理。

1. 全课主线

智能体在环境中进行表示、学习、决策与推理。

关键概念

  • 本项不作为单独节点展开。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 导航任务可从 MDP 状态行动奖励，也可从 CSP 变量约束建模。

老师强调点

  • 建模优先是贯穿全课重点。

易错点

  • 易错：只写公式不解释变量含义。

考试可能考法

  • 模块比较题。

2. 机器学习重点

特征、损失、优化、泛化是四件套。

关键概念

  • 本项不作为单独节点展开。

公式/规则

  • w←w−η∇Loss

典型例子

  • 给训练误差和测试误差判断过拟合。

老师强调点

  • 特征和泛化反复强调。

易错点

  • 易错：把神经网络当脱离损失的黑箱。

考试可能考法

  • SGD 和过拟合辨析。

3. MDP/RL 重点

MDP 是概率奖励序列决策，RL 是未知模型下的交互学习。

关键概念

  • 本项不作为单独节点展开。

公式/规则

  • V(s)=max_a Q(s,a)
  • Q←(1−η)Q+η[r+γmaxQ']

典型例子

  • 穿越火山程序是理论到代码的核心样例。

老师强调点

  • V/Q 和价值迭代是高频重点。

易错点

  • 易错：策略评估和价值迭代混淆。

考试可能考法

  • 价值迭代和 Q-learning 手算。

4. CSP/BN/Logic 重点

三者都做推理，但表示方式不同。

关键概念

  • 本项不作为单独节点展开。

公式/规则

  • BN: P=∏CPD
  • Logic: KB∧¬φ⇒□

典型例子

  • CSP 适合约束满足，BN 适合不确定性，Logic 适合规则证明。

老师强调点

  • 表示方式决定推理算法。

易错点

  • 易错：把因子图、BN 和逻辑规则混为一谈。

考试可能考法

  • 模块比较与易错辨析。

# 课程纪要重点：全课老师强调重点

本章核心主题：本页把课堂笔记纪要中的老师强调点转成总复习重点，帮助把 PPT 知识与考试答题语言统一。

老师重点：不要只背算法名，要先说清楚建模对象：特征、状态、变量、概率关系或逻辑命题是什么。

## 1. 全课主线

### 1.1 二级知识点

- 核心概念：智能体在环境中进行表示、学习、决策与推理。
- 公式/规则：本节以概念/流程为主
- 典型例子：导航任务可从 MDP 状态行动奖励，也可从 CSP 变量约束建模。
- 老师重点：建模优先是贯穿全课的重点。
- 易错点：易错：只写公式不解释变量含义。
- 考点：模块比较题。

### 1.2 先建模

- 类型：老师重点
- 说明：任何算法题先问输入、输出、状态/变量/特征/命题。

### 1.3 再计算

- 类型：老师重点
- 说明：公式和算法在模型清楚后执行。

### 1.4 最后解释

- 类型：考点
- 说明：结果要回到问题语义。

### 1.5 模块比较

- 类型：考点
- 说明：说清 ML、MDP、CSP、BN、Logic 的适用场景。

## 2. 机器学习重点

### 2.1 二级知识点

- 核心概念：特征、损失、优化、泛化是四件套。
- 公式/规则：w←w−η∇Loss
- 典型例子：给训练误差和测试误差判断过拟合。
- 老师重点：特征和泛化反复强调。
- 易错点：易错：把神经网络当脱离损失的黑箱。
- 考点：SGD 和过拟合辨析。

### 2.2 特征决定上限

- 类型：老师重点
- 说明：模型只能利用特征中存在的信息。

### 2.3 SGD

- 类型：公式/推导
- 说明：沿负梯度方向逐步降低损失。

### 2.4 泛化

- 类型：老师重点
- 说明：训练好不代表测试好。

### 2.5 验证/测试

- 类型：易错点
- 说明：验证集调参，测试集最终评估。

## 3. MDP/RL 重点

### 3.1 二级知识点

- 核心概念：MDP 是概率奖励序列决策，RL 是未知模型下的交互学习。
- 公式/规则：V(s)=max_a Q(s,a)；Q←(1−η)Q+η[r+γmaxQ']
- 典型例子：穿越火山程序是理论到代码的核心样例。
- 老师重点：V/Q 和价值迭代是高频重点。
- 易错点：易错：策略评估和价值迭代混淆。
- 考点：价值迭代和 Q-learning 手算。

### 3.2 MDP 五元组

- 类型：老师重点
- 说明：S,A,T,R,γ 是入口。

### 3.3 V/Q 区分

- 类型：易错点
- 说明：V 看状态，Q 看状态-行动。

### 3.4 价值迭代

- 类型：公式/推导
- 说明：Bellman 最优更新。

### 3.5 Q-learning

- 类型：公式/推导
- 说明：未知 T/R 时从经验更新 Q。

## 4. CSP/BN/Logic 重点

### 4.1 二级知识点

- 核心概念：三者都做推理，但表示方式不同。
- 公式/规则：BN: P=∏CPD；Logic: KB∧¬φ⇒□
- 典型例子：CSP 适合约束满足，BN 适合不确定性，Logic 适合规则证明。
- 老师重点：表示方式决定推理算法。
- 易错点：易错：把因子图、BN 和逻辑规则混为一谈。
- 考点：模块比较与易错辨析。

### 4.2 CSP

- 类型：老师重点
- 说明：变量、域、约束；搜索与剪枝。

### 4.3 BN/HMM

- 类型：老师重点
- 说明：随机变量、DAG、CPD；概率分解和动态推理。

### 4.4 filtering/smoothing

- 类型：易错点
- 说明：前者看当前以前，后者用未来证据。

### 4.5 Logic

- 类型：老师重点
- 说明：语法、语义、模型、蕴涵、CNF 和归结。

- 必背：V/Q、训练/测试、AC-3、BN 分解、CNF/归结
- 需要做题：模块比较、公式手算、算法流程

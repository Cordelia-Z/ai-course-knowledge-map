二、层级知识结构

按“一级知识点 → 概念/公式/例子/重点/易错/考点”整理。

1. 监督学习问题设置

给定输入 x 和标签 y，学习一个能泛化到新样本的预测函数。

关键概念

  • 训练数据：由样本 x 与标签 y 组成，用于学习参数。
  • 分类：输出离散类别。
  • 回归：输出连续数值。
  • 预测函数：从输入映射到输出的函数，是学习目标。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 邮件分类、情感分类、房价预测。

老师强调点

  • 先明确输入、输出和标签。

易错点

  • 易错：分类和回归区别在输出类型。

考试可能考法

  • 解释监督学习、分类和回归。

2. 特征提取

把原始对象变成模型可计算的向量。

关键概念

  • 特征 φ(x)：描述样本的重要属性，是模型看到世界的方式。
  • 特征向量：多个特征组成的数值向量。

公式/规则

  • 本项不作为单独节点展开。

典型例子

  • 情感分类中，“good”出现次数可以是特征。

老师强调点

  • 特征表示决定模型能不能学。

易错点

  • 易错：原始数据不等于有效表示。

考试可能考法

  • 给任务设计特征。

3. 线性预测器

用权重向量衡量每个特征贡献。

关键概念

  • 权重 w：表示每个特征对预测结果的贡献。
  • 线性边界：在线性特征空间中用超平面划分类别。

公式/规则

  • score(x)=w·φ(x)

典型例子

  • 积极词权重大，出现积极词会提高正类得分。

老师强调点

  • 要能解释权重和特征的语义。

易错点

  • 易错：权重不是标签，而是特征贡献。

考试可能考法

  • 手算 score。

4. 损失最小化与 SGD

定义错得多严重，再用梯度逐步更新参数。

关键概念

  • 损失函数：预测与真实答案之间的惩罚。

公式/规则

  • min_w Σ_i Loss(x_i,y_i,w)
  • w ← w − η∇Loss

典型例子

  • 每看一个样本，模型根据错误方向稍微调整权重。

老师强调点

  • SGD 是考试最可能手算的更新机制。

易错点

  • 学习率太大可能震荡，太小收敛慢。

考试可能考法

  • 手算一轮 SGD 更新。

# Lecture 2：Machine Learning I

本章核心主题：本章建立监督学习基本链条：数据、特征、线性预测器、损失函数和随机梯度下降。

老师重点：机器学习不是模型魔法，而是表示、预测、损失和优化的组合。

## 1. 监督学习问题设置

### 1.1 二级知识点

- 核心概念：给定输入 x 和标签 y，学习一个能泛化到新样本的预测函数。
- 公式/规则：本节以概念/流程为主
- 典型例子：邮件分类、情感分类、房价预测。
- 老师重点：先明确输入、输出和标签。
- 易错点：易错：分类和回归区别在输出类型。
- 考点：解释监督学习、分类和回归。

### 1.2 训练数据

- 类型：核心概念
- 说明：由样本 x 与标签 y 组成，用于学习参数。

### 1.3 分类

- 类型：核心概念
- 说明：输出离散类别。

### 1.4 回归

- 类型：核心概念
- 说明：输出连续数值。

### 1.5 预测函数

- 类型：核心概念
- 说明：从输入映射到输出的函数，是学习目标。

## 2. 特征提取

### 2.1 二级知识点

- 核心概念：把原始对象变成模型可计算的向量。
- 公式/规则：本节以概念/流程为主
- 典型例子：情感分类中，“good”出现次数可以是特征。
- 老师重点：特征表示决定模型能不能学。
- 易错点：易错：原始数据不等于有效表示。
- 考点：给任务设计特征。

### 2.2 特征 φ(x)

- 类型：核心概念
- 说明：描述样本的重要属性，是模型看到世界的方式。

### 2.3 特征向量

- 类型：核心概念
- 说明：多个特征组成的数值向量。

### 2.4 人工特征

- 类型：图示/流程
- 说明：由人设计规则提取，如词频、长度、颜色。

### 2.5 表示质量

- 类型：老师重点
- 说明：特征决定模型上限。

## 3. 线性预测器

### 3.1 二级知识点

- 核心概念：用权重向量衡量每个特征贡献。
- 公式/规则：score(x)=w·φ(x)
- 典型例子：积极词权重大，出现积极词会提高正类得分。
- 老师重点：要能解释权重和特征的语义。
- 易错点：易错：权重不是标签，而是特征贡献。
- 考点：手算 score。

### 3.2 权重 w

- 类型：核心概念
- 说明：表示每个特征对预测结果的贡献。

### 3.3 score

- 类型：公式/推导
- 说明：w 与 φ(x) 的内积。

### 3.4 决策规则

- 类型：图示/流程
- 说明：分类选最高得分类别，回归输出数值。

### 3.5 线性边界

- 类型：核心概念
- 说明：在线性特征空间中用超平面划分类别。

## 4. 损失最小化与 SGD

### 4.1 二级知识点

- 核心概念：定义错得多严重，再用梯度逐步更新参数。
- 公式/规则：min_w Σ_i Loss(x_i,y_i,w)；w ← w − η∇Loss
- 典型例子：每看一个样本，模型根据错误方向稍微调整权重。
- 老师重点：SGD 是考试最可能手算的更新机制。
- 易错点：学习率太大可能震荡，太小收敛慢。
- 考点：手算一轮 SGD 更新。

### 4.2 损失函数

- 类型：核心概念
- 说明：预测与真实答案之间的惩罚。

### 4.3 经验风险

- 类型：公式/推导
- 说明：训练集上损失总和或平均。

### 4.4 梯度

- 类型：公式/推导
- 说明：函数上升最快方向，负梯度用于下降。

### 4.5 SGD

- 类型：公式/推导
- 说明：用样本或小批量估计梯度并更新权重。
